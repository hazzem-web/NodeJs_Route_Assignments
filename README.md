# 🟢 Node.js Route — Assignments

A collection of progressive backend assignments built throughout the **Node.js Route** learning path — starting from JavaScript fundamentals all the way up to full REST APIs with MongoDB, Sequelize, JWT authentication, and advanced database operations.

Each assignment lives in its own folder with a dedicated README.

---

## 📦 Assignments Overview

| # | Assignment | Stack | Description |
|---|---|---|---|
| 1 | [Assignment1](./Assignment1) | JS | JavaScript fundamentals — 15 exercises + LeetCode bonus |
| 2 | [Assignment2](./Assignment2) | Node.js | Core modules practice — `path`, `fs`, `events`, `os` |
| 3 | [Assignment3](./Assignment3) | Node.js | Pure HTTP server — CRUD with no frameworks |
| 4 | [Assignment4](./Assignment4) | Node.js + Express | JSON file-based User API + ERD diagram |
| 5 | [Assignment5](./Assignment5) | Express + MySQL | Raw SQL queries — products, suppliers, sales & permissions |
| 7 | [Assignment6&7](./Assignment6&7) | Express + Sequelize + MySQL | Social platform API — users, posts, comments |
| 8 | [Assignment8](./Assignment8) | Express + MongoDB Native | Books/Authors/Logs — aggregation, capped collections, indexing |
| 9 | [Assignment9](./Assignment9) | Express + MongoDB + Mongoose | Notes API — JWT auth, pagination, aggregation pipeline |

---

## 📈 Learning Progression

```
JS Basics → Node.js Core → Raw HTTP → Express → SQL → Sequelize → MongoDB Native → Mongoose + JWT
```

| Phase | Assignments | What was learned |
|---|---|---|
| JavaScript Core | A1 | Types, closures, array methods, destructuring |
| Node.js Internals | A2, A3 | `fs`, `path`, `events`, `os`, Event Loop, HTTP server |
| Express Basics | A4 | REST API structure, JSON file storage |
| Raw SQL | A5 | DDL/DML, JOINs, permissions with `mysql2` |
| ORM | A7 | Sequelize, associations, hooks, paranoid delete, upsert |
| MongoDB | A8 | Native driver, aggregation pipelines, capped collections |
| Mongoose + Auth | A9 | JWT, bcrypt, ownership guards, pagination, `$lookup` |
